Política de Privacidad

En Tuiteraz valoramos la privacidad de los usuarios. Esta política explica cómo recopilamos, usamos y protegemos la información.

Información que recopilamos

  • Datos de inicio de sesión mediante Google (como correo electrónico y nombre).
  • Ubicación aproximada del dispositivo para proporcionar información del clima.
  • Datos de uso dentro de la aplicación, como elementos marcados como favoritos.

Uso de la información

La información recopilada se utiliza para:

  • Permitir el inicio de sesión del usuario.
  • Guardar preferencias como elementos favoritos.
  • Mostrar información personalizada, como el clima basado en la ubicación.

Servicios de terceros

La aplicación utiliza servicios de terceros que pueden recopilar información:

  • Google (para autenticación y servicios relacionados).
  • Supabase (para almacenamiento y gestión de datos).

Almacenamiento de datos

Los datos se almacenan de forma segura mediante servicios en la nube y no se venden ni comparten con terceros con fines comerciales.

Permisos

La aplicación puede solicitar acceso a la ubicación del dispositivo únicamente para mostrar información del clima. No se accede a archivos, fotos u otros datos del dispositivo.

Seguridad

Tomamos medidas razonables para proteger la información del usuario.

Cambios a esta política

Nos reservamos el derecho de actualizar esta política en cualquier momento.
